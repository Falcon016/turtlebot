# Turtle Bot (static site mock)

A lightweight static mock of a **professional download website** for **Turtle Bot** (OpenClaw wrapper), with a new **green turtle** brand.

## Open it

- Open `turtleclaw-ui/index.html` in a browser.

## Included

- `index.html`: Download landing page (hero + downloads + features + install + changelog)
- `styles.css`: Dark professional theme with turtle-green accents (no red)
- `assets/turtleclaw-mascot.svg`: Chunky outlined turtle mascot (replaces the lobster)
- `assets/turtleclaw-mark.svg`: Compact app icon / mark
